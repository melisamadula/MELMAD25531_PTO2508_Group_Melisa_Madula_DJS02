# DJS02 – Web Component: Podcast Preview

## Description 📜:

We were tasked to create a landing page for a Podcast App. The page needed (to be responsive) to have desktop, tablet and mobile view. Usability was limited to a certain amount of features, as this is a project we are to build upon - until it becomes a fully usable Podcast App, with full functionality.

## Task 🖊️:

- To use: Functional Programming, OOP, SOLID, abstractions, modules and modular coding.
- To use: Vanilla JavaScript.
- To use: customElements.define() to enhance modularity, promote reuse, and reduce code duplication across the app.
- To: build upon our previous project (DJS01) code solution, by adding and editing code to fufil our current project requirements.

## Techonolgy 💻:

- HTML
- CSS
- JavaScript

## Features 🎨:

- Podcast cards.
- Within the cards: Podcast - title, seasons, genre and last updated date.
- Modal: once having clicked on a Podcast card, you get more information like - description and an extended list of seasons with amount of episodes per season.
- Scroll bars: for both the Podcast landing page and modal.
- Button: to exit modal, plus red hover on that "x" (exit) button.
- Blur effect: once modal is opened.
- Responsiveness (changing screen viewing visuals per screen size): desktop, tablet, mobile.
